Term ABS
Description of term: Антиблокировочная система — система, предотвращающая блокировку колёс транспортного средства при торможении.
Основное предназначение системы — сохранение устойчивости и управляемости автомобиля
Term SSH
Description of term: SSH — сетевой протокол прикладного уровня, позволяющий производить удалённое управление операционной системой и туннелирование TCP-соединений.
Схож по функциональности с протоколами Telnet и rlogin, но, в отличие от них, шифрует весь трафик, включая и передаваемые пароли.
