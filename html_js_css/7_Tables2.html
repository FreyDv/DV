    border-collapse: separate; Свойство border-collapse может иметь 3 значения:
      collapse, separate и inherit
      collapse -  сжимает границы в одну, выбирая ячейку таблицы в качестве основной.
      separate -  все разные границы складываются друг с другом образуя толщину без сжимания

    border-spacing - свойство работает только с в случае border-collapse: separate
      задаёт, какое расстояние, если оно есть, отображается между разделенными границами.
      border-spacing: 5px горизонталь  5px вертикаль 

    border - свойство которое позволяет конфигурировать
      параметры линии которая выступает в виде границы ячеек и таблиц
      border: 1px ТОЛЩИНА  solidТИП
